# AWS_Fullstack_Event_RSVP
This is the source for the Event_RSVP full stack application which is deployed and managed on AWS.

Below is the architecture of the app:
Frontend: HTML, CSS, JavaScript
Backend: Node.js(AWS Lambda)
Database: MySQL(RDS) + DynamoDB
Host: S3 + CloudFront
